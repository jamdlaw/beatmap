## Setup

To activate the Python virtual environment:

- **For Git Bash:**

      source venv/Scripts/activate

- **For Command Prompt (cmd.exe):**

      venv\Scripts\activate.bat

- **For PowerShell:**

      venv\Scripts\Activate.ps1
